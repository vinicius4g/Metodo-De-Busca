# Metodo-De-Busca
Buscador de padrão em textos  Um buscador de padrão em textos utilizando o método shiftand, feito com Java. Utilizando a tabela ASCII para definir o alfabeto. O usuario informa o caminho do arquivo de texto que deseja procurar o padrão, informa o padrão, em seguida é criado um clone do arquivo original com os padrões encontrados, eles estaram MAIUSCULOS.
